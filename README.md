# Pizza-sales-Report
The project name is pizza sales Report we use tools SQL and Power BI
Overview

This project demonstrates an end-to-end data analytics workflow, starting from data ingestion into a MySQL database to performing SQL-based analysis, designing a Power BI dashboard, and generating insights in a final report.
The goal is to showcase the ability to work with relational databases, perform analytical queries, visualize key metrics, and communicate data-driven conclusions.Objectives

Import and manage datasets using MySQL

Clean and transform data using SQL

Run analytical SQL queries to uncover trends and insights

Build an interactive Power BI dashboard

Summarize key findings in a professional reportTools & Technologies

SQL Analysis

Descriptive statistics (counts, sums, averages)

Trend analysis using time-based queries

Segmentation and grouping

Advanced queries using window functions, CTEs, and subqueries
